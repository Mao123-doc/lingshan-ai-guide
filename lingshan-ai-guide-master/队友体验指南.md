# 🏯 灵山胜境 AI 数字人导游 — 零基础体验指南

> **你不需要懂任何编程知识，照着下面的步骤一步步做就行。**
> **遇到任何问题，截图发给我！**

---

## 第 1 步：安装 Node.js（必须，只做一次）

**什么是 Node.js？** 你可以把它理解成"让这个项目能跑起来的底座"，装一次就行。

1. 打开你的浏览器（Chrome 或 Edge），在地址栏输入：**https://nodejs.org**，然后回车
2. 页面中间有两个按钮，**点左边那个绿色的**，上面写着 "LTS"（版本号可能是 20.x 或 22.x，无所谓）
3. 会下载一个文件叫 `node-v22.x.x-x64.msi`，下载完后在浏览器底部能看到它
4. **双击**这个下载好的文件
5. 弹出一个安装窗口，点 **"Next"**（下一步）
6. 继续点 **"Next"**
7. 继续点 **"Next"**（不要改任何选项，一路默认）
8. 最后点 **"Install"**（安装）
9. 等进度条跑完，点 **"Finish"**（完成）

> ✅ **检查一下装好了没：** 按键盘上的 `Win 键`（就是那个 Windows 图标键）+ `R`，会弹出一个小窗口，输入 `cmd` 然后回车。会弹出一个黑色窗口，在里面输入 `node --version` 然后回车。如果出现 `v22.x.x` 这样的数字，就说明装好了。关掉黑色窗口。

---

## 第 2 步：安装 Python（必须，只做一次）

**什么是 Python？** 语音播报功能需要它，也是装一次就行。

1. 在浏览器地址栏输入：**https://www.python.org/downloads/**，回车
2. 页面中间有一个**黄色大按钮**，上面写着 "Download Python 3.x.x"，点它
3. 下载完后，**双击**这个文件
4. ⚠️ **非常重要！** 弹出来的安装窗口**最下面**有一个复选框叫 **"Add Python to PATH"**（把 Python 加到系统路径），**一定要勾上！**（点一下那个小方框，让它打上勾）
5. 勾好之后，点上方的 **"Install Now"**（现在安装）
6. 等进度条跑完，点 **"Close"**（关闭）

> ✅ **检查一下装好了没：** 跟上面一样，`Win + R` → 输入 `cmd` → 回车，黑色窗口里输入 `python --version` 回车。如果出现 `Python 3.x.x` 就对了。

---

## 第 3 步：下载项目

1. 打开浏览器，在地址栏输入：**https://github.com/freeze614/lingshan-ai-guide**，回车
2. 页面上有一个**绿色的按钮**，上面写着 **"<> Code"**，点它
3. 弹出来的小菜单里，点最下面那个 **"Download ZIP"**
4. 浏览器会下载一个压缩包，文件名类似 `lingshan-ai-guide-master.zip`
5. 下载完后，在浏览器底部点一下这个文件（或者去"下载"文件夹里找到它）
6. **右键点击**这个 zip 文件 → 选择 **"全部解压缩"**（或 "Extract All"）
7. 弹出来的窗口里，直接点底部的 **"提取"**（或 "Extract"）
8. 解压完后会多出来一个文件夹，名字是 `lingshan-ai-guide-master`

> 📁 **记住这个文件夹在哪！** 最好把它放在桌面上，方便找。

---

## 第 4 步：启动项目（以后每次只需要做这一步！）

1. **打开**刚才解压出来的 `lingshan-ai-guide-master` 文件夹
2. 在里面找到 **`启动数字人.bat`** 这个文件（图标像一个小齿轮）
3. **双击**它
4. 会弹出一个黑色窗口，它会自动做以下事情：
   - 检查你的 Node.js 和 Python 装好没
   - 如果是第一次运行，会自动下载项目需要的东西（5-10 分钟，仅此一次）
   - 启动服务
5. 等待黑色窗口里出现 **"✨ 系统已就绪！正在打开浏览器..."** 这句话
6. 你的浏览器会自动打开，进入「灵小禅」数字人导游页面！
7. 🎉 **开始聊天吧！**

> 📌 **注意：** 启动过程中会额外弹出 3 个黑色小窗口，**不要关掉它们！** 把它们最小化就行。

---

## 第 5 步：跟数字人聊天

浏览器打开后，你会看到一个古风界面，中间是数字人「灵小禅」。

**打字提问：**
- 页面底部有一个输入框，在里面打字
- 比如输入"灵山大佛有多高？"
- 按键盘上的 **Enter（回车）** 发送
- 灵小禅会用文字 + 语音回答你

**语音输入（用嘴说话）：**
- 点输入框左边的 **🎤 麦克风按钮**
- 浏览器会弹窗问"是否允许使用麦克风"→ 点 **"允许"**
- 然后直接说话就行了
- ⚠️ 必须用 Chrome 或 Edge 浏览器才支持

**让数字人说话：**
- 打开右上角的 **"自动播放"** 开关
- AI 回答后会自动用语音读出来

**连续对话模式：**
- 打开右上角的 **"连续对话"** 开关
- 听完回答后自动等你下一句话

**快捷提问：**
- 直接点输入框上方的彩色标签（"灵山大佛有多高？"等）

---

## 🛑 用完怎么关

回到项目文件夹，双击 **`停止服务.bat`**，所有窗口会自动关闭。

## ⚠️ 常见问题

| 遇到的情况 | 怎么办 |
|-----------|--------|
| 双击 bat 后提示"未安装 Node.js" | 你没装 Node.js，回到第 1 步 |
| 双击 bat 后提示"未安装 Python" | 你没装 Python，回到第 2 步（注意勾选 Add to PATH） |
| 第一次启动等了很久 | 正常，要下载东西。耐心等，不要关窗口 |
| 浏览器打开了但是页面加载不出来 | 等 10 秒刷新一下，服务还在启动中 |
| 数字人回答文字但不说话 | ① 检查电脑音量 ② Chrome 地址栏左边小锁 → 允许声音 ③ 打开"自动播放"开关 |
| 点了麦克风没反应 | 必须用 Chrome 或 Edge 浏览器，并且在弹窗里点了"允许" |
| 三个服务窗口报错 | 截图发给我 |
