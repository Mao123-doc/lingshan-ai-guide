## 三、实施步骤

### 阶段 1：精美 Live2D 数字人（~2h）
- 用 Agnes Image API 生成汉服少女角色立绘
- 搭建 PixiJS 渲染引擎
- 实现分层精灵系统（头发后/身体/衣服/手臂/头发前/表情层）
- 实现 viseme 嘴型映射（5种嘴型）
- 实现情感表情驱动（6种情感）
- 实现待机动画（呼吸+微动+眨眼）
- 实现语音驱动口型同步

### 阶段 2：知识库真正实现（~1.5h）
- mammoth 解析上传的 docx 文档
- 自动分块+关键词提取
- 写入知识库索引
- 知识库文档列表管理（增删）
- RAG 准确率测试框架（50题）

### 阶段 3：真多模态 + 真情感分析（~1h）
- Agnes Image API：拍照→识景→讲解
- 对话 LLM 情感分类管线
- 感受度报告数据真实化

### 阶段 4：语音全链路 + 管理端实时化（~1.5h）
- Whisper 服务端 ASR 备选
- TTS viseme 时间轴生成
- WebSocket 实时推送管理端数据
- 数据大屏自动刷新

### 阶段 5：文档 + 演示视频脚本（~1h）
