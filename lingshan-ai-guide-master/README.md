# 🏯 灵山胜境 AI 数字人导游系统 v2.2

> AI Digital Human Tour Guide System for Ling Shan Sacred Land

## 项目简介

基于AI数字人技术的智能景区导览系统，为"灵山胜境"5A级景区构建7×24小时在线的"AI导游"。具备**Live2D WebGL 数字人渲染**、**多模态交互**（语音对话闭环、表情、口型同步）、**RAG智能问答**（查询改写 + 重排序 + 三层检索）、**个性化路线推荐**、**GPS位置感知**，以及完整的**管理后台**（数据大屏、知识库管理、游客感受度分析）。

### 🆕 v2.2 新特性
- 🤖 **Live2D WebGL 数字人**：pixi-live2d-display 驱动 hibiki 模型，实时口型同步 + 表情切换 + 空闲动作
- 🎤 **语音对话闭环**：AI回答后自动播放语音，支持连续对话模式（免提）
- 📍 **GPS位置感知**：查找附近景点、距离提示、到达引导
- 🔍 **RAG增强**：LLM查询改写 + Reranker 重排序，准确率大幅提升
- ⭐ **游客满意度评分**：对话内即时评价，数据驱动服务优化
- 📱 **PWA支持**：可添加到主屏幕，接近原生App体验

## 系统截图

### 游客端
- 🏠 **首页**：景区介绍、热门问题、GPS查找附近景点
- 💬 **智能问答**：Live2D数字人实时对话，自动语音播放、连续对话模式

- 🎭 **数字人管理**：4套外观风格 + 3种语音，独立搭配，实时预览
- 📝 **感受度报告**：情感分布饼图、热门话题排行、景点关注度排名、AI分析总结

## 🚀 快速开始（零基础也能跑）

### 环境要求
- **Node.js** >= 18（下载：https://nodejs.org ，点绿色 LTS 按钮）
- **Python** >= 3.8（下载：https://www.python.org/downloads/ ，⚠️ 安装时勾选 "Add Python to PATH"）

### 一键启动

1. 打开 https://github.com/freeze614/lingshan-ai-guide → 点绿色 **Code** 按钮 → **Download ZIP** → 解压到桌面
2. 双击 `启动数字人.bat`
3. 首次运行自动安装依赖（5-10 分钟），之后只需 30 秒

浏览器会自动打开 → 直接开始跟数字人「灵小禅」对话！

| 页面 | 地址 | 说明 |
|------|------|------|
| 🏯 游客端 | http://localhost:8000 | 数字人问答、路线推荐 |
| 🔧 管理后台 | http://localhost:8000/admin/login | 数据大屏、知识库管理 |
| | 账号 `admin` 密码 `lingshan2026` | |

### 关闭服务

双击 **`停止服务.bat`** 一键关闭所有服务。

> 📖 详细图文指南见 `队友体验指南.md`
